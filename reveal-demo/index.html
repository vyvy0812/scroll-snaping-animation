Khái niệm cốt lõi của snap scroll

GIẢI THÍCH
Snap Point
Là một vị trí cố định trên thanh scroll.
Khi bạn dừng scroll gần đó, trình duyệt sẽ tự động "hút" về đúng điểm đó.

Giống như nam châm — thả tay là dính vào.

Website fullscreen

Scroll xuống section tiếp theo — tự nhảy vào đầu section, không bị cắt nửa chừng.

Snap points trong demo bên trái
Code CSS
/* 1. Bật snap trên container */
.container {
  scroll-snap-type: y mandatory;
  /* mandatory = bắt buộc snap */
}

/* 2. Đặt snap point tại mỗi section */
.section {
  scroll-snap-align: start;
  /* start = snap vào đầu section */
}
2 loại snap

mandatory — bắt buộc

Dừng ở đâu cũng bị hút về snap point gần nhất. Không thoát được.

proximity — gần mới hút

Chỉ hút khi bạn dừng gần snap point. Dừng xa thì không bị kéo.
